# Serverless-Spark-ETL-Pipeline-on-AWS
AWS Glue ETL Pipeline Lab
Overview
This project implements an end-to-end automated data pipeline using AWS services. The pipeline processes customer review data stored in Amazon S3, performs transformations using AWS Glue (PySpark), and generates analytical outputs.
The workflow is fully automated using AWS Lambda, which triggers the Glue job whenever a new file is uploaded to the S3 landing bucket.
________________________________________
Architecture Overview
The pipeline consists of the following components:
1.	Amazon S3 (Landing Bucket)
Stores raw input data (reviews.csv). 
2.	AWS Lambda
Triggered automatically when a new file is uploaded to S3.
Starts the AWS Glue ETL job. 
3.	AWS Glue (ETL Job)
Performs data transformation and analysis using PySpark. 
4.	Amazon S3 (Processed Bucket)
Stores processed data and query results. 
________________________________________
Pipeline Workflow
1.	Upload reviews.csv file to the S3 landing bucket:
s3://handsonfinallanding-lakshmi/
2.	S3 triggers the Lambda function:
start_glue_job_trigger
3.	Lambda invokes the AWS Glue job:
process_reviews_job
4.	Glue job performs: 
o	Data cleaning 
o	Data transformation 
o	SQL-based analytics 
5.	Results are stored in:
s3://handsonfinalprocessed-lakshmi/
________________________________________
Data Transformations Performed
The AWS Glue job performs the following transformations:
•	Convert rating column to integer and replace null values with 0 
•	Convert review_date to proper date format 
•	Replace null values in review_text with default text 
•	Standardize product_id to uppercase 
________________________________________Analytical Queries Implemented
The following Spark SQL queries were executed:
1. Average Rating per Product
Calculates average rating and total reviews for each product.
________________________________________
2. Date-wise Review Count
Counts number of reviews submitted per day.
________________________________________
3. Top 5 Most Active Customers
Identifies customers who submitted the highest number of reviews.
________________________________________
4. Overall Rating Distribution
Shows count of each rating (1–5 stars).
